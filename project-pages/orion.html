Orion (CalHacks Winner)
Orion is an AI-powered incident command center that unifies situational awareness, routing, and communication into a single dashboard to help responders act faster and safer.
Project Details / Background
In California, more than 26 million residents live under wildfire threat, and incident commanders often rely on outdated systems like radios, spreadsheets, texts, and maps to coordinate responses. Through conversations with volunteers and responders, we saw how critical minutes were lost switching tools instead of fighting fires. We envisioned Orion as an AI-powered command center that unifies situational awareness, routing, and communication in a single dashboard, helping responders act faster, smarter, and safer.
Orion is a real-time wildfire operations dashboard that combines live incident data, weather models, and AI agents to deliver end-to-end response coordination. It features 3D maps with wind-aware fire spread prediction, responder tracking, and dynamic dispatch routing. Background Python uAgents (Fetch.ai) monitor incidents, compute fire-spread risk, and propose optimal resource allocation in real time. VAPI voice control enables hands-free commands like “Dispatch two crews to Pine Ridge” or “Show predicted wind shifts.” Supabase provides sub-second data synchronization, while Mapbox renders containment progress and route animations. Integrated Telegram alerts deliver updates instantly to on-ground teams.
